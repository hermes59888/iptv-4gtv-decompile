切换线路或 API 设置后，系统会自动清除相关缓存，下次播放将重新获取地址。
使用建议
  • 遇到播放卡顿或无法加载时，可尝试切换线路并清空缓存。
  • 线路 0 (Hinet CDN) 在部分地区宽带环境下表现更优；线路 1 (官方) 整体兼容性最佳。
  • 切换线路后系统会自动清除该频道的播放缓存，确保新线路立即生效。
  • 「清空缓存」按钮会清除所有频道的播放地址缓存，不影响其他配置。
  • 「自动提取最高解析度」开启后，无论选择线路0还是1，都会尝试返回最高码率流。
  • 「使用 TV API」开关可改变部分频道获取播放地址的接口。
  • 「自动切换 API 与 线路」开启后，播放失败时会自动尝试其他线路和 API 组合，提高成功率。
