配置说明
  • 多代理支持: 可以新增多个 VLESS/VMess/Trojan 节点，系统会根据延迟自动選擇最佳节点代理
  • 自动選擇: 啟用后系统会定期测试所有节点代理的延迟，并自动切换到最快的可用节点代理
  • 延迟测试间隔: 設定自动测试代理延迟的间隔分钟数，仅在自动選擇啟用时生效
  • 动态端口: Xray 内部端口自动适配 Docker 映射端口 (目前: {{ xray_port }})
  • Xray 版本管理: 可以手动更新 Xray 核心版本，更新后需要重新启动代理
  • Xray 官网: https://github.com/XTLS/Xray-core/releases
  • 导出/导入节点配置: 支持将所有 VLESS/VMess/Trojan 节点配置导出为 JSON 档案，或从 JSON 档案导入恢复配置（导入将完全替换现有配置）。
  • 完整流媒体代理: 啟用后，所有流媒体数据都会通过服务器转发，播放端无需特定网络即可观看。
